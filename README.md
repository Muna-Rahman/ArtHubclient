#  ArtHub

## Purpose
ArtHub is a full-stack digital art marketplace where artists can showcase and sell artwork, while collectors can purchase art, manage their collections, and leave reviews.

## Live URL
https://arthub-mauve.vercel.app

## Key Features
- Secure authentication with Better Auth
- Role-based dashboards (Admin, Artist, Collector)
- Stripe payment integration
- Artwork upload and management
- Purchase-based reviews and transaction history
- Membership plans with upload limits
